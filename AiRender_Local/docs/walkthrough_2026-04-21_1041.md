# Walkthrough: Instalador Inteligente KreanRender Local

He completado la implementación del sistema de instalación profesional y guiado para el Add-in local.

## Cambios Realizados

### 1. Nueva Interfaz de Usuario (XAML)
He rediseñado la ventana de `KreanRenderLocal` para aplicar un flujo de trabajo lineal:
- **Botón de Análisis**: Detecta la potencia del PC.
- **Barra de Progreso**: Muestra el avance real de la descarga.
- **Bloqueo Inteligente**: Los botones se activan secuencialmente (1 -> 2 -> 3).

### 2. Detección de Hardware (C# + WMI)
El Add-in ahora es capaz de:
- Identificar la GPU y su **VRAM**.
- Medir la **RAM total** del sistema.
- **Bloquear la instalación** si el equipo tiene menos de 8GB de RAM (evitando cuelgues).
- Recomendar el modo **CPU** o **GPU** según la tarjeta detectada.

### 3. Motor de Instalación (Python)
He creado el script `install_engine.py` que realiza las siguientes tareas de forma aislada:
- Descarga una versión de **Python Embebido** (no necesita instalación en Windows).
- Configura **pip** y las variables de entorno.
- Instala **PyTorch** (optimizado para CPU o GPU según el análisis previo).
- Descarga las librerías necesarias (`diffusers`, `opencv`, etc.).

## Cómo Probarlo

1. **Abrir Revit**: Ve a la nueva pestaña **DabaDev** y lanza **Render Local**.
2. **PASO 1**: Pulsa "Analizar mi PC". Verás que detecta tu MX350 y te recomienda el modo adecuado.
3. **PASO 2**: Selecciona una carpeta (ej: en un disco externo). El programa verificará que tienes los 12GB necesarios.
4. **PASO 3**: Pulsa "Instalar". Verás la barra de progreso avanzar mientras se descarga el motor.
5. **Renderizar**: Una vez instalado, se habilitará el botón de **Renderizar**.
